# Auth — реализация на NestJS + ORM Prisma

- Реализована : **регистрация**, **авторизация**, **сброс пароля**, **выход из системы**, в зависимости от роли пользователя.

- **refresh** и **access** токены.

- et-Cookie.

- Подтвержление регистрации по почте через пакет **nodemailer**, работает опционально через включение из файла **.env**, настройки там же + в папке **EmailVerificationModule**.

- Защита от ботов через механизм **throttler**, файлы в папке **security**, работает опционально через.включение из файла **.env**.

- Анализ ботов, в БД создата таблица: **suspiciousActivityLog**.

- Роли: **USER**, **ADMIN** через **enum** в файле **schema.prisma**, доступы к модулям исходя из роли пользователя.

- Конфиги для docker

- и т.д.

## Установка

Из docker так и любым другим способом, например через npm i.
